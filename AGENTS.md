## 一、最高优先级执行协议

以下规则优先于本文档其他内容及任何长规范；如出现冲突，以本节为准。

1. **不确定就停止**：缺 PDF、缺页码、缺溯源表、数据无法验证时，不得编造，不得继续生成投资结论。正确做法是输出"无法验证，停止在当前阶段"。
2. **报告必须先过门禁**：新标的先筛选 → 通过后做数据溯源 → `check_report_ready.py` 通过后才能写报告。
3. **核心数字必须有来源**：现金、负债、利润、现金流、CAPEX、股本、市值、估值倍数，必须标注文件/年份/页码或溯源表编号。
4. **禁止口头完成**：凡"生成/更新/写入/修改"，必须实际改文件，并在最终回复列出文件路径。
5. **禁止自动 git**：除非用户明确要求，不得 `git commit`、`git push`、`git reset`、`git checkout --`。
6. **冲突先报告**：发现规则冲突时先指出并请求确认，不得自行选择有利规则。
7. **免责**：本项目为投资研究工具，不构成投资建议；未经核查的数据不得用于买卖建议。

## 二、长规范调用门禁

任务命中以下场景时，必须先读取对应长规范，并在执行前回复"已读取文件清单"。未列出清单，不得继续执行。

| 任务类型 | 必读规范 |
|---|---|
| 生成/更新投资分析报告 | `docs/ai-rules/01-报告生成SOP.md` + `docs/ai-rules/02-数据溯源门禁.md` + `readPDF.md` |
| 提取/核对财报数据 | `readPDF.md` + `docs/ai-rules/02-数据溯源门禁.md` + `数据溯源/数据准确性保障流程.md` |
| 更新网页导航/索引（用户明确要求时） | `docs/ai-rules/03-网页索引更新.md` |
| 模拟组合/每日监控/决策追踪 | `docs/ai-rules/04-模拟组合数据规则.md` |
| 记录数据错误/复盘 | `docs/ai-rules/99-错误案例库.md` |

**执行前回执模板**（命中上表时必须先输出）：

```
已读取：<规范文件清单>
本任务适用门禁：<阶段/校验脚本>
当前阶段：<阶段编号>
暂不适用：<规则及原因>
```

## 三、幻觉防护红线

禁止：

- 用第三方网页数据冒充年报 S 级数据
- 无 PDF 页码却写"来自年报"
- 把估算值写成确定值
- 为完成报告而补齐不存在的数据
- 跳过溯源表直接写报告
- 未经用户确认就继续处理未通过筛选的新标的
- 用旧报告数字作为新报告唯一依据

校验脚本（关键步骤必须运行并贴返回码）：

```bash
python scripts/check_report_ready.py {股票代码}    # 报告写作前门禁
python scripts/check_index_sync.py                  # 网页索引同步校验
python scripts/validate_simulation_data.py          # 模拟组合数据一致性
```

## 五、项目定位

港股/A股个股投资分析模板系统，知识管理型项目。核心产出：标准化投资分析报告、数据溯源表、风险监控、决策追踪。

- 目标用户：个人投资者（资金十来万港币）
- 市场：港股（优先）、A股
- 流派：纯硬收息 / 价值发现 / 烟蒂股 / 关联方资源型

## 六、版本与文件规范

- 版本格式 `V{主}.{次}.{修订}`，**当前 V5.5.14**。全仓"当前版本"只能有一处声明。
- 模板文件需版本管理（旧版归档到 `归档版本/`，变更记入 `docs/CHANGELOG.md`）。
- 具体公司分析报告**不做版本后缀**，同名直接覆盖，只留一份最新版。
- 脚本头部需 `# -*- coding: utf-8 -*-` 与版本注释。

### 数据源分级

| 等级 | 数据源 | 限制 |
|------|--------|------|
| S | 年报/港交所披露易原文附注 | 必须用原文数字，禁止估算 |
| A | 业绩公告/官网 | 财务数据需 S 级验证 |
| B | Bloomberg/Wind/东方财富 | 不得用于核心财务数据 |
| C | 财经媒体/研报 | 仅供参考 |
| D | AI 估算 | 禁止用于投资决策 |

## 七、估值标准速查

| 市场 | 剔除净现金估值 | 股息率 | PB |
|------|---------------|--------|-----|
| 港股（熊市） | FCF倍数 < 3 倍 | > 6% | < 0.6 |
| A股 | PE/FCF倍数 < 15 倍 | > 4% | — |

### 关键术语

| 术语 | 含义 |
|------|------|
| 金龟股 | PB<0.6、股息率>6%、央国企背景的高股息标的 |
| 烟蒂股 | 净现金/市值>2倍的极度低估标的 |
| FCF | 自由现金流 = 经营现金流 − 资本开支 |
| 可支配FCF | 扣除少数股东权益后归母自由现金流 |

## 八、投资哲学核心（龟龟投资理论）

- **防御优先**：追求绝对防御，熊市90%以上时间跑赢指数，收益结合回撤看。
- **普通人假设**：视线1-2年，回避大宗商品等高波动变量，只做确定性高的"小学生数学题"。
- **反对压票思维**：比较内在价值与市场报价，估值达公允即因机会成本切换。
- **基金模式持仓**：约10只个股，单只≤10%（上限20%），不押注少数公司。
- **风险记忆**：记录历史最大回撤，注重安全边际与低波动。
