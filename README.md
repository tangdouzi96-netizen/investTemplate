# 个股分析标准模版

> **四流派投资体系 + 龟龟投资理论 + 风险记忆 + 数据质量硬约束 + 财报数据使用规范 + 实战笔记整合 + AI辅助深度分析**
>
> 版本以 [AGENTS.md](AGENTS.md) 为准（当前 V5.5.18）。

---

## 📋 项目简介

本模板是港股/A股个股投资的**深度体检工具**，整合四大投资流派（纯硬收息/价值发现/烟蒂股/关联方资源型），专注于**安全边际、真实估值、管理人诚信**三大核心。

**核心原则**：
- 一级市场思维："10元买瓜，自己吃也值15元"
- 烟蒂股哲学：赚价值修复的钱，不赚成长的钱
- 保守主义：宁可错过，不可做错

## 🚀 快速开始（5分钟初筛）

分析任何标的前，先执行 **5分钟快速初筛**：

| 检查项 | 排除标准 | 你的标的 |
|--------|---------|----------|
| **① 价格位置** | 距52周高点<30%（已反弹>70%） | □ |
| **② 负债扫描** | 财务费用>5亿（高有息负债） | □ |
| **③ 现金覆盖** | 现金<市值20%（安全边际不足） | □ |
| **④ 盈利连续性** | 连续2年经营现金流为负 | □ |

**快速通过信号**：
- 🏆 财务费用<0 + 现金>市值50% → 无有息负债企业
- 🐢🍊 股价距52周低点<10% + T0覆盖率>80% → 标准烟蒂股
- 🟠 价格位置不佳但估值合理 → 试探建仓+网格加仓

**不通过？** → 放入观察列表，等待改善

**通过？** → 打开 `个股分析标准模版.md` 进行深度分析

**通过但位置不佳？** → 试探建仓（10%仓位）+ 设定梯度加仓计划

## 🎯 四流派投资体系

| 流派 | 核心特征 | 适用标的 | 分析重点 |
|------|---------|---------|----------|
| **🛡️ 纯硬收息** | 高股息（>6%）、稳定现金流 | 电信/公用事业/REITs | 派息可持续性 |
| **🔍 价值发现** | 低估（PE<10）、轻资产、FCF充裕 | 消费/制造/服务 | 真实盈利能力 |
| **🚬 烟蒂股** | 净现金>市值、靠近历史低点 | 港股小盘/国企 | 清算价值、管理人诚信 |
| **🔗 关联方资源** | 国企背景、母公司支持 | 物业/基建/公用事业 | 利益输送风险 |

## 📊 V5.5.12 核心升级（观察仓框架+成长性陷阱+模拟投资组合整合版）

1. **观察仓 vs 试探建仓框架**：0仓位纯观察（如达势30PE等2年）vs 2-5%试探建仓
2. **成长性估值陷阱警示**：30PE成长股2年修复到15PE，不如5PE+8%股息价值股
3. **时间成本量化计算**：成长股需要31%回报才能打平价值股16%股息+风险补偿
4. **"被认为衰退但实际稳健"识别**：市场偏见带来的极端低估机会
5. **模拟投资组合（50万港币）**：4只标的建仓方案，考虑每手数量，**每日股价监控**，预期年化19%收益

## 📊 V5.5.10 核心升级（数据质量硬约束整合版）

| 模块 | 功能 | 实战价值 |
|------|------|----------|
| **数据质量硬约束系统** | YAML模板→校验脚本→内嵌校验表三阶段流程 | 杜绝数据错误，强制使用最新完整年度年报 |

| **龟龟投资理论整合** | 防御型价值投资 + 风险记忆 + 普通人假设 | 基于实战经验提炼的防御性投资哲学 |
| **数据错误作废机制** | 发现错误→立即停止→重新计算→更新报告 | 建立错误处理流程，确保分析质量 |
