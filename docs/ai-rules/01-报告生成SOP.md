# 01 · 投资分析报告生成 SOP

> 本文件为长规范。生成或更新投资分析报告前必须先读取本文件，并在回复中列出"已读取规范"。
> 上位规则见根目录 `AGENTS.md` 的"最高优先级执行协议"。

---

## 阶段门禁总览

```
阶段0 任务识别
   ↓
阶段1 标的状态判断（新标的先 6 关筛选，等用户确认）
   ↓
阶段2 数据溯源门禁  →  python scripts/check_report_ready.py {代码}（返回1则停止）
   ↓
阶段3 报告写作（只引用溯源表数据）
   ↓
阶段4 网页索引同步（默认非强制，见 03 号规范）
   ↓
阶段5 完成验收（列产物清单）
```

任一阶段失败必须停止，不得跳步。

---

## 阶段0：任务识别

- 用户只是提问 → 只回答问题，不生成报告。
- 用户要求生成/更新投资分析报告 → 进入阶段1。

## 阶段1：标的状态判断

| 场景 | 处理 |
|------|------|
| 已有报告 | 读取旧报告 + 最新财报 + 现有溯源表，判断是否需要更新 |
| 新标的 | 执行下方 6 关定量筛选，输出筛选表，**等用户确认**再继续 |

未通过筛选的新标的：停止，告知卡在第几关，不得继续生成完整报告。
已分析标的清单以 `07-分析输出/index.md` 为准（不另维护重复列表）。

### 6 关定量筛选

| # | 维度 | 通过标准 | 不通过 |
|---|------|---------|--------|
| 1 | 护城河代理（三选二） | ①毛利率5年稳定/上升+行业前五 ②ROIC 5年均值>12%+前三 ③市占率前三 | 三项全不满足→筛掉 |
| 2 | 盈利质量 | FCF/净利润 5年均值 > 70% | <70%→筛掉 |
| 3 | 管理层红线（全过） | ①ROE 5年均值>12%（非金融）②近3年无造假/违规 ③大股东近3年未大规模减持(>5%) | 任一不过→筛掉 |
| 4 | 财务安全 | 净现金 − 有息负债 > 0 | 不过→筛掉（REITs/金融/基建豁免） |
| 5 | 股息可持续 | ①连续5年分红 ②分红率≤80% ③股息率>4%(A)/>6%(港) | 任一不过→筛掉 |
| 6 | 估值闸门 | 剔除净现金FCF倍数<3倍(熊市) 或 PE<15倍(A股) | 不过→筛掉 |

**CAPEX 预警**（不直接筛掉，配合 ROIC 分档）：CAPEX/经营现金流 5年均值 <30% 绿灯；30-50% 正常；50-70% 预警（ROIC>15%可接受，<10%深度质疑）；>70% 严重预警（回溯5年趋势）。

### 改进框架七大缺口（V5.5.14+，报告需覆盖）

管理层能力正向评估 / 护城河耐久性 / 盈利质量拆解 / 竞对深度对比 / 激励与利益一致性 / 表外项目与或有负债 / 股东回报全面分析。

## 阶段2：数据溯源门禁（硬门禁）

必须存在并达标：`数据溯源/{公司}_{代码}_数据溯源表.md`

运行校验：

```bash
python scripts/check_report_ready.py {股票代码}
```

返回码：0 通过 / 2 通过但有⚠️待复核 / 1 失败禁止写作。

必须把脚本输出贴给用户。返回 1 时停止，先补溯源表。

详见 `docs/ai-rules/02-数据溯源门禁.md`。

## 阶段3：报告写作

- 输出路径：`07-分析输出/{公司名称}_{股票代码}_投资分析报告.md`
- 同名报告直接覆盖，不保留历史版本（公司报告只留一份最新版）。
- 标题下第一屏必须有 `**一句话结论**：<当前状态+估值位置+操作建议>`。
- 所有核心财务数字只能来自溯源表；需要新数字必须回阶段2补充。
- 核心计算必须展示：公式 + 代入数值 + 结果 + 数据来源（页码/溯源表编号）。
- 禁止创建"快速参考""速览"等辅助文件。

## 阶段4：网页索引同步

默认非强制。仅当用户提到"网站/VitePress/侧边栏/首页展示"时执行，详见 `docs/ai-rules/03-网页索引更新.md`。

## 阶段5：完成验收

最终回复必须列出：

- 报告文件路径
- 溯源表文件路径
- 使用的 PDF/公告文件
- 核心数据是否均有页码（贴 `check_report_ready.py` 返回码）
- 未解决红线/待验证项数量
- 是否同步了索引（若执行了阶段4）
- 是否未执行 git（默认不自动提交）

## 港股 vs A股 差异化估值标准

| 市场 | 剔除净现金估值 | 股息率 | PB |
|------|---------------|--------|-----|
| 港股（熊市） | FCF倍数 < 3 倍 | > 6% | < 0.6 |
| A股 | PE 或 FCF倍数 < 15 倍 | > 4% | — |

## 报告更新频率

- 股价日常波动、触及买卖点、市场情绪：**不更新报告**，用监控清单跟踪。
- 新财报、重大并购/重组、管理层变更、发现数据错误：**必须更新报告**。
