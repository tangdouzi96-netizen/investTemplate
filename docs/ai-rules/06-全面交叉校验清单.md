# 06 · 全面交叉校验清单

> **职责**：规定溯源表与报告在「PDF 提取」之外还必须完成的**全部**交叉校验项。
> 上位规则见 `AGENTS.md`；PDF 读取细则见 `readPDF.md` §4；门禁脚本见 `scripts/check_report_ready.py`。
> **版本**：V1.1 · 2026-06-05

---

## 为什么需要本文件

`check_report_ready.py` 能阻断：缺页码、缺 BS/CF 勾稽、A 区占位、来源造假、验证汇总年份分层违规。

但它**不能**单独发现：
- 溯源表内部自洽、但与市场/常识严重偏离（如茅台股息率 2.21% vs 行情 4.1%）
- 只取 p2 年度预案、漏读中期/特别分红
- 终审只核对「报告=溯源表」、未做合理性校验

**禁止把「脚本返回 0」等同于「数据已全面验证」。** 第4层合理性校验由高级模型终审完成。

---

## 四层校验体系（全部必做）

```
┌─────────────────────────────────────────────────────────────┐
│ 第1层 报表内部勾稽     BS/CF/PL 平衡、B区公式回算、0值确认   │ → 脚本阻断
├─────────────────────────────────────────────────────────────┤
│ 第2层 年报内部交叉     摘要↔正式表、分红章节↔p2、次年比较列  │ → 自检块 + 脚本警告
├─────────────────────────────────────────────────────────────┤
│ 第3层 外部独立源比对   WS 按年份分层（见 §3.2）              │ → 验证汇总表 + 脚本
├─────────────────────────────────────────────────────────────┤
│ 第4层 报告终审合理性   报告↔溯源表 + 衍生指标市场合理性      │ → 高级模型验收记录
└─────────────────────────────────────────────────────────────┘
```

---

## 第1层 · 报表内部勾稽（脚本阻断）

| 检查项 | 公式/方法 | 阻断条件 |
|--------|----------|----------|
| BS 平衡 | 资产总计 = 负债合计 + 所有者权益合计 | 缺 ✅ |
| CF 平衡 | 经营+投资+筹资+汇率 ≈ 现金净增 | 缺 ✅ |
| PL 勾稽 | 收入−成本−税金−三费±财务 ≈ 营业利润 | 差异 >5% 须说明 |
| B 区回算 | 毛利率、ROE、FCF、支付率等 | 与 A 区不一致 |
| 0 值科目 | BS 行存在但余额=0 | 未确认（WebSearch 或附注） |

## 第2层 · 年报内部交叉（自检块必填）

### 2.1 摘要页 ↔ 正式报表

摘要页营收/归母/EPS/总资产/经营 CF = 正式报表对应行（容差 ≤0.1%）。

### 2.2 分红结构完整性（高发遗漏区）

**禁止只读 p2「年度利润分配预案」。** 必须读「利润分配或资本公积金转增预案」整章（约 p30–p35）。

| 检查 | 方法 |
|------|------|
| 中期/特别分红 | 搜索「中期利润分配」「回报股东特别分红」 |
| 每股股息合计 A42 | 年度 A40 + 中期/特别 A41 = 合计 |
| 分红总额 A44 | 与「累计派发现金红利 XX 亿元」一致 |
| 股利支付率 | A44÷A11 与年报披露比对（容差 ≤1pp） |

> **案例**：贵州茅台仅取 p2 的 27.99 元 → 股息率 2.21%；补中期后 A42=51.95 → 4.1%。

### 2.3 次年比较列（🔗）

2022–2024 年可用**下一年年报**「上年同期」列验证，在验证汇总表标 🔗。

### 2.4 有息负债 / 受限资金

有息负债空值须确认；受限资金查附注「所有权或使用权受到限制的资产」。

## 第3层 · 外部独立源比对（WS 分层）

### 3.1 科目分级（A / B / C）

| 级别 | 科目 | 主表 WS 列 | 验证汇总表 |
|:--:|------|------------|------------|
| **A 类** | 营收、归母净利、经营 CF、CAPEX、**每股股息合计**、**分红总额** | 禁止 📝/⚠️/空白（若汇总表已按 §3.2 通过，主表可 📝） | **按年份分层（§3.2）** |
| **B 类** | 现金、总负债、资产总计 | 组内至少一行 ✅/🔗/📄 | 建议列入汇总表 |
| **C 类** | 费用、BS 子项、存货等明细 | 允许 📝（须已过勾稽） | 可选 |

**WS 合格标记**：✅（WebSearch/公告，差值≤1%）、🔗（次年比较列）、📄（摘要页）、📝（仅 PDF+勾稽，**仅 2021 年 A 类允许**）。

### 3.2 A 类 · 按年份分层（项目定稿规则）

| 年份 | 要求 | 合格标记 | 不合格 |
|------|------|----------|--------|
| **2025** | 尽量外部独立确认 | ✅ 或 📄（公告级）；🔗 可过但**建议改 ✅** | 📝、⚠️、空白 |
| **2022–2024** | 第二证据即可 | ✅、🔗、📄 | 📝、⚠️、空白 |
| **2021** | **不要求 WebSearch** | 📝、✅、🔗、📄 均可 | ⚠️、空白 |

说明：
- **2025** 是投资决策主依据，优先 WebSearch/交易所公告打 ✅。
- **2022–2024** 追溯重述多、网站口径乱，🔗 次年比较列差值=0 即可。
- **2021** 仅要求 PDF 页码 + 勾稽；验证汇总表标 📝 即达标，**不强制搜索**。
- 不适用年份可写 `—`（如历史年份无股息率行情）。

### 3.3 搜索关键词模板

| 科目 | 模板 |
|------|------|
| 营收/归母 | `{公司} {代码} {年} 营业收入 归母净利润` |
| 经营 CF / CAPEX | `{公司} {代码} {年} 现金流量表 购建固定资产` |
| 每股股息合计 | `{公司} {代码} {年} 分红 每股 中期 特别` |
| 分红总额 | `{公司} {代码} {年} 现金分红 总额` |

### 3.4 验证汇总表（必填）

格式参照 `数据溯源/中国食品_00506_数据溯源表.md`：

```markdown
## WebSearch 验证汇总

| 科目 | 2021 | 2022 | 2023 | 2024 | 2025 | 来源 |
| 营业收入(亿) | 1062📝 | 1241🔗 | 1477🔗 | 1709✅ | 1688✅ | 东方财富/次年比较列 |
| 每股股息合计(元) | 21.68📝 | 43.59🔗 | 49.98🔗 | 51.51✅ | 51.95✅ | 公告/年报利润分配章 |
```

**门禁**：
- 缺验证汇总区块 → 警告（返回 2）
- A 类科目 2022–2025 年份分层违规 → **阻断（返回 1）**
- 2025 年仅 🔗 无 ✅ → 警告（建议补 ✅）

### 3.5 衍生指标（C 类，写入汇总或 B 区）

| 指标 | 公式 | 外部参照 | 异常阈值 |
|------|------|----------|----------|
| 股利支付率 | A44÷A11 | 年报利润分配章 | >2pp |
| **股息率** | **A42÷股价（C01）** | 行情/券商 | **>30% 须排查** |
| FCF | A33−A34 | B01 | 不一致 |

> **股息率铁律**：禁止用 A40 年度预案；必须用 A42 全年合计。

## 第4层 · 报告终审合理性（高级模型必做）

### 4.1 报告 ↔ 溯源表

报告核心数必须引用溯源编号；股息率用 A42，支付率用 A45 或 A44÷A11。

### 4.2 合理性交叉（终稿验收必填）

| 指标 | 检查 |
|------|------|
| 股息率 | A42÷股价 vs 行情，偏差 ≤30% |
| 股利支付率 | vs 年报利润分配章 |
| 五年累计分红 | = A44 各年加总 |
| 同业数据 | 标 S/A/C，禁冒充 S 级 |

### 4.3 验收记录模板

```markdown
### 合理性交叉校验（第4层）

| 指标 | 报告值 | 外部/内部参照 | 偏差 | 结果 |
| 股息率 | 4.10% | 行情~4.1% | <5% | ✅ |
```

**任一项超阈值且未解释 → 不得标「终稿完成」。**
