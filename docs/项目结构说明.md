# 项目结构说明

> **版本以 [`AGENTS.md`](../AGENTS.md) 为准（当前 V5.5.18）。** 本页描述仓库目录布局与入口关系；详细工作流见 [`docs/工作流程图.md`](工作流程图.md)。

## 一、当前主链（日常用这个）

```
investTemplate/
├── 个股分析标准模版.md              # 🔥 人读导航入口
├── AGENTS.md                        # AI 最高执行协议（版本唯一声明处）
├── README.md / index.md             # 项目说明 / VitePress 首页
│
├── template/                        # 📁 现版分析方法论（01～10 章 + 草稿壳）
│   ├── 00-草稿骨架.md               # AI 填报告草稿用
│   └── 01～10-*.md                  # 第一～十章方法论
│
├── 数据溯源/                        # 财报溯源表（数字唯一可信来源）
├── readPDF.md                       # PDF 提取技术规范
│
├── 07-分析输出/                     # 投资分析报告
│   ├── index.md                     # 目录页（现版 / 既往分区）
│   ├── *_投资分析报告.md            # ✅ 现版报告（17 份，根目录）
│   └── 既往分析报告/                # 📦 归档报告（11 份，非当前决策依据）
│
├── 08-决策追踪/                     # 模拟组合、决策记录、trades.csv
├── 模拟持仓/                        # VitePress 持仓仪表盘页面
├── 07-标的追踪/                     # 标的代码追踪
│
├── docs/                            # 项目文档
│   ├── CHANGELOG.md                 # 版本变更日志
│   ├── 工作流程图.md                # 全景导航
│   ├── 项目结构说明.md              # 本文件
│   └── ai-rules/                    # 报告 SOP、门禁、交叉校验（01～06、99）
│
├── scripts/                         # check_report_ready.py、索引同步、监控 CI 等
├── config/                          # risk_management.yaml、validation YAML
├── .vitepress/ + index.md           # VitePress 站点
└── 归档版本/                        # 历史模版全文 + 早期分散框架
```

### 快速入口

| 你想做什么 | 打开哪里 |
|------------|----------|
| 人工深度分析 | [`个股分析标准模版.md`](../个股分析标准模版.md) → `template/01～10` |
| AI 写报告 | [`AGENTS.md`](../AGENTS.md) → `docs/ai-rules/01～06` → `check_report_ready.py` |
| 查报告结论 | `07-分析输出/{公司}_{代码}_投资分析报告.md` |
| 查数字来源 | `数据溯源/{公司}_{代码}_数据溯源表.md` |
| 看版本历史 | [`docs/CHANGELOG.md`](CHANGELOG.md) |
| 本地预览网站 | `npm run docs:dev`（默认 5173，占用则顺延） |

## 二、现版 vs 归档（别混）

| 类型 | 路径 | 说明 |
|------|------|------|
| **现版分析框架** | `template/01～10` | V5.5.14 分层重构后的章节方法论，由导航页链入 |
| **现版投资报告** | `07-分析输出/*.md`（根目录） | 17 份，同名覆盖，登记在三处网页索引 |
| **既往投资报告** | `07-分析输出/既往分析报告/` | 11 份历史/旧版，网站侧边栏单独分组 |
| **历史模版全文** | `归档版本/个股分析标准模版_V*.md` | 各版本完整快照 |
| **早期分散框架** | `归档版本/早期框架文档(V5.5.0之前)/` | 旧 `01～06` 编号模块，**已退役** |

> 网站侧边栏：**「✅ 现版报告」** 与 **「📦 既往分析报告」**、**「📦 历史框架（V5.5.0 前）」** 三组分离，避免与现版混列。

## 三、早期框架（01～06 · 仅归档参考）

V5.5.0 之前，内容按编号模块散落在根目录 `01-筛选框架/` … `06-附录案例/`。  
V5.5.14 起合并为 `template/` + `个股分析标准模版.md` 导航；旧模块整体迁入：

`归档版本/早期框架文档(V5.5.0之前)/`

| 旧模块 | 归档路径 | 现版替代 |
|--------|----------|----------|
| 01-筛选框架 | `…/01-筛选框架/` | `template/01`、`template/02` |
| 02-数据清洗 | `…/02-数据清洗/` | `template/01` 数据质量硬约束 |
| 03-估值模型 | `…/03-估值模型/` | `template/06` |
| 04-决策分析 | `…/04-决策分析/` | `template/07` |
| 05-策略框架 | `…/05-策略框架/` | `template/08～10` |
| 06-附录案例 | `…/06-附录案例/` | `07-分析输出/` 实战报告 |

**不作为当前分析主链**；网站仅通过侧边栏「历史框架（归档）」提供只读入口。

## 四、AI 与校验

| 路径 | 用途 |
|------|------|
| `AGENTS.md` | AI 最高优先级协议 |
| `docs/ai-rules/01～06、99` | 报告 SOP、溯源门禁、网页索引、模拟组合、DeepSeek 提示词、交叉校验、错误案例 |

## 五、文件命名规范

- **现版导航**：`个股分析标准模版.md`（始终覆盖，无版本后缀）
- **历史模版**：`归档版本/个股分析标准模版_V{版本}_{名称}.md`
- **现版报告**：`07-分析输出/{公司}_{代码}_投资分析报告.md`（无版本后缀，同名覆盖）
- **溯源表**：`数据溯源/{公司}_{代码}_数据溯源表.md`
