## ★ 公司概览（易漏小节①）

- **主营业务**：物业管理服务——基础物业管理（住宅/商业/公建）、增值服务（社区增值/工程服务/案场服务）、城市服务。中海物业是中国物业管理协会副会长单位，连续多年位列中国物业服务百强企业 TOP10。
- **控股结构**：中国海外集团有限公司（「中海集团」/COLI 体系）为直接大股东，持有 **56.07%** 已发行股份（2025AR p96）；经中建股份→中建集团穿透，最终权益约 **61.24%**（2025AR p96）。最终实控人为中国建筑集团有限公司（央企）。
- **核心业务线**（按 2025 年报分部披露）：
  - 物业管理服务（基础物管费收入，核心基本盘）
  - 增值服务（社区增值、工程服务、案场顾问等）
  - 城市服务（政府/公建物业外包）
- **上市与市场**：香港联交所主板，股票代码 02669.HK，2015 年 10 月由 COLI 分拆上市。
- **行业定位**：轻资产物业管理公司（适用标准 FCF/CAPEX/毛利率框架；**勿套用保险 P/EV 或银行 PB 主公式**）。

## 4. 负债与财务安全

> **★ 受限资金（易漏小节②）**：2021–2023 年综合财务状况表及附注均无受限资金披露（A-NOTE）；2024 年受限银行存款 0.09 亿（2024AR p123 附注 26）；2025 年受限银行存款 0.13 亿（2025AR p129 附注 26）。受限金额极小，不影响整体流动性判断。

### 负债结构（2025 年末，A 区数据）

| 项目 | 金额 | 来源 |
|------|------|:--:|
| 资产总额 | **129.59 亿** | A15 / 2025AR p128-129 |
| 总负债 | **70.28 亿** | A16 / 2025AR p128-130 |
| 资产负债率 | **54.2%** | A19 / ➖衍生 |
| 银行借款（全部为流动负债） | **0.50 亿** | A20 / 2025AR p129 附注33 |
| 租赁负债（流动+非流动） | **0.96 亿** | 2025AR p128-129 附注16(b)：5407万+4187万 |
| 贸易应付款等（经营负债） | **约 69.3 亿** | 总负债 70.28亿 − 借款 0.50亿 − 租赁 0.96亿（➖） |
| 现金及银行结余 | **62.71 亿** | A14 / 2025AR p129 附注27 |
| 其中：受限银行存款 | **0.13 亿** | A20b / 2025AR p129 附注26 |
| **净现金** | **62.21 亿** | ➖ A14 − A20 = 62.71 − 0.50 |

### 负债安全性分析

1. **有息负债极低**：银行借款仅 0.50 亿（A20），占总资产 0.4%，几乎可以忽略。5 年趋势：2021=0 → 2022=0.68亿¹ → 2023=0.56亿 → 2024=0.50亿 → 2025=0.50亿，持续维持在极低水平。
2. **经营负债为主**：总负债 70.28 亿中扣除银行借款 0.50 亿后约 69.78 亿为经营负债（贸易应付款、合同负债/预收物业费、应付职工薪酬、应交税费等），属无息负债，体现轻资产模式下对上下游的占款能力。
3. **净现金充裕**：净现金 62.21 亿占资产 48.0%，约为市值（C02≈115.7亿 RMB）的 **54%**。极端情景下现金垫厚，但**不等于现价已低于净现金**（现价对应市值仍高于净现金）。
4. **流动性充足**：现金 62.71 亿 vs 银行借款 0.50 亿，覆盖率 > 100 倍。即使考虑全部经营负债的短期到期，现金比率也远超安全线。

> ¹ 2021–2022 为港币亿元，2023–2025 为人民币亿元，跨年对比存在币种差异。

**财务安全结论**：极低有息负债 + 充裕净现金 + 轻资产 CAPEX = 财务安全度极高。唯一关注点是物业公司的经营负债中合同负债（预收物业费）占比大，这部分非真正「负债」而是待确认收入。

**2025 年利润下滑根因**：营收 +6.7% 但毛利 −3.4%（A03），毛利率从 16.6% 压缩至 15.0%（A04），降幅 1.6 个百分点。归母净利 −9.7%（B02）大于毛利降幅，说明费用端亦有压力（销售行政费用 A05 −2.3% 基本持平，但减值损失扩大——2025 金融资产减值 1.28 亿，见 PL 勾稽验算）。

### 5.2 毛利率趋势分析

毛利率从 2021 年 17.4% 持续下行至 2025 年 15.0%。可能原因：
- **人工成本刚性上升**：物业管理人员薪酬 + 社保合规成本 ↑
- **第三方外拓项目毛利率低于关联方**：外拓竞标压价拉低综合毛利率
- **增值服务结构变化**：低毛利工程服务占比上升
- **分部结构**：物业管理服务占收益 **78.4%**（2025AR p42），在管面积 **4.776 亿㎡**（+8.0%）；关联方项目毛利率通常高于第三方外拓竞标项目（2025AR p39-43 定性披露）

5 年中 4 年经营 CF > 归母净利的 80%，表明利润现金含量较好。2023 年经营 CF/净利 > 100%（营运资金改善），2024–2025 回落但仍健康。

### 5.4 应计与存货

- **存货**：物业管理属服务行业，存货（材料/未结算服务合同）占比极低，不存在存货减值风险。
- **应收款**：贸易应收 **28.77 亿**（2025AR p128 附注22），占营收 **19.2%**（➖）；合同资产 4.77 亿。关联方应收需关注账期，但无单项重大减值披露。
- **商誉**：综合财务状况表**无单独商誉科目**；无形资产 2.88 亿（2025AR p128 附注17）。会计政策明确同一控制下合并**不确认商誉**（2025AR p214）；外拓并购商誉风险有限，但仍须跟踪无形资产减值。

**利润质量结论**：毛利率趋势向下是核心隐忧，但经营现金流含金量较好，利润有现金流支撑。

### 7.3 中海物业的特殊性

- **优势**：央企背景 + 极低有息负债 + 关联方（中海地产）财务稳健（对比碧桂园服务关联方暴雷风险）
- **劣势**：科技标签和市场关注度低于万物云；增值服务毛利率不及华润万象生活的商业运营；品牌溢价不及绿城服务
- **护城河差异**：关联方订单是核心壁垒（中海地产年销售规模 2000–3000 亿，持续交付在管面积），但这也意味着独立性不足——中海地产销售下行直接冲击物管增量

## 10. 护城河持久性

### 10.1 护城河来源

| 护城河类型 | 强度 | 说明 |
|------|:--:|------|
| **关联方资源壁垒** | ★★★★☆ | 中海地产（COLI）年销售额 2000–3000 亿港元级，为中海物业持续提供在管面积「粮仓」。第三方物管公司无法获取此订单流 |
| **央企品牌与信用背书** | ★★★☆☆ | 政府/公建类项目招标中，央企背景有加分；业主信任度高于民营物管 |
| **规模与成本优势** | ★★★☆☆ | 在管面积 **4.776 亿㎡**、进驻 **164 城**（2025AR p28-29），规模带来调度效率 |
| **转换成本** | ★★☆☆☆ | 业委会更换物管公司有一定流程成本，但非不可逾越（尤其住宅项目） |
| **品牌溢价** | ★★☆☆☆ | 不如绿城服务/万科物业，属中上水平 |

### 10.2 护城河可复制性

- **关联方资源难以复制**：中海地产的土储和年销售规模是 30 年积累的，竞争对手无法短期复制
- **但也不是永久壁垒**：中海地产自身销售规模受行业下行影响，2022–2025 年全国商品房销售面积持续下降，中海地产亦难独善其身
- **第三方外拓可部分替代**：如果第三方外拓能力足够强（如万物云模式），可减轻关联方依赖

### 10.3 护城河半衰期估计

保守估计现有关联方资源壁垒的半衰期约 **5–8 年**：5 年内中海地产的交付基本盘仍在（基于当前土储），但若销售持续下行，8 年后新增在管面积可能大幅萎缩。第三方外拓的成败是决定半衰期延长或缩短的关键变量。

> **★ 护城河侵蚀（易漏小节④）**：必答题「什么能侵蚀这个护城河？」

### 10.4 护城河侵蚀信号 ⚠️

| 侵蚀因素 | 威胁程度 | 监控指标 | 现状 |
|------|:--:|------|:--:|
| **地产竣工与销售下行** | 🔴 高 | 全国房屋竣工面积增速；中海地产合约销售额 | 2022–2025 持续下行 |
| **关联方在管面积收缩** | 🔴 高 | 中海地产年交付面积；中海物业新增在管面积中关联方占比 | 2025 营收增速已降至 6.7%（A01） |
| **第三方竞标压价** | 🟡 中 | 外拓项目中标毛利率 vs 关联方项目毛利率 | 综合毛利率已降至 15.0%（A04） |
| **科技物管/机器人替代** | 🟡 中 | 万物云等竞对的科技降本效果；AI+IoT 物管解决方案渗透率 | 长期趋势，短期影响有限 |
| **人工与社保成本刚性** | 🟡 中 | 最低工资年增幅；社保缴费基数调增 | 每年约 5–10% 的人工成本上涨 |
| **物业费限价政策** | 🟡 中 | 各城市物业费政府指导价调整 | 部分城市已有限价但整体可控 |
| **业主维权意识增强** | 🟢 低 | 业委会成立比例；换物业事件频率 | 缓慢上升但非爆发式 |

**最核心的侵蚀逻辑**：中海物业的护城河本质是「中海地产的开发业务 → 物管在管面积」的转化链条。当地产开发这条上游河流水量减少时，下游物管的护城河自然干涸。这不是物业管理能力的问题，而是结构性冲击。第三方外拓和增值服务是两条可能的「引水渠」，但能否完全替代中海地产的「主河道」仍有待验证。

## 13. 情景分析与买入区间

### 13.1 情景设定（以 2025 年数据为基准）

| 情景 | 概率 | 核心假设 | FCF（亿） | 目标 FCF 倍数 | 企业价值（亿） | 每股价值（HKD）¹ |

| **基准** | 45% | 利润持平、FCF≈10亿；估值维持 5×FCF+净现金 | 10.33 | 5× | 115.7→**≈4.0 HKD** | +6% |

> ¹ 情景目标价基于 `(倍数×FCF+净现金)÷股数÷1.07`（RMB 口径），起点价 **3.77 HKD**（C01）。² 乐观 FCF=13.61亿（A23，2023 峰值）。

### 13.2 买入价公式（易漏小节⑤，必填）

**公式**（龟龟港股标准，轻资产物管适用）：

```
买入价(HKD/股) = (目标FCF倍数 × FCF − 净现金) ÷ 股数 ÷ 汇率(RMB→HKD)
```

**代入数值**（龟龟港股熊市标准：目标 FCF 倍数 = 3 倍）：

| 参数 | 数值 | 来源 |
|------|------|:--:|
| 目标 FCF 倍数 | **3 倍** | 龟龟港股熊市标准（AGENTS.md），理由：轻资产物管合理估值区间 3–5× FCF，取保守下沿 |
| FCF | **10.33 亿**（2025 实际） | A23 |
| 净现金 | **62.21 亿**（A14 − A20，2025 末 RMB） | ➖ A14=62.71亿, A20=0.50亿 |
| 股数 | **32.84 亿股** | A30 |
| 汇率（RMB→HKD） | **~1.07** | C 级（2026-06 近似；1 RMB≈1.07 HKD） |

**计算**：

```
买入价 = (3 × 10.33 − 62.21) ÷ 32.84 ÷ 1.07
       = (30.99 − 62.21) ÷ 32.84 ÷ 1.07
       = −31.22 ÷ 32.84 ÷ 1.07
       = −0.95 ÷ 1.07
       ≈ −0.89 HKD/股
```

**正常化 FCF 口径**（2023–2025 三年均值 11.90 亿）：

```
买入价(正常化) = (3 × 11.90 − 62.21) ÷ 32.84 ÷ 1.07
                = (35.70 − 62.21) ÷ 32.84 ÷ 1.07
                = −25.51 ÷ 32.84 ÷ 1.07
                ≈ −0.73 HKD/股
```

**解读**：公式结果为负，说明在 **3 倍 FCF** 假设下，经营业务隐含价值低于账上净现金——这是**资产垫厚**而非「现价已免费」。**龟龟买入门槛**对应目标市值 ≈ 62.21 + 3×10.33 = **93.2 亿 RMB** → 目标价 **≈2.65 HKD/股**。

**买入区间（终审）**：**2.0–2.7 HKD**（约 3×剔净现金 FCF 至接近净现金/股 2.03 HKD）。**现价 3.77 HKD** 高于区间上限，**不满足**建仓条件。

## 17. 决策流程与仓位建议

### 17.1 买入逻辑声明（支持方）

1. **净现金垫厚**：62.21 亿净现金（A14−A20），每股约 **2.03 HKD** 现金底价，下行有硬资产支撑。

3. **央企信用**：中海系背景，关联方暴雷风险远低于民营物管（对比碧桂园服务）。
4. **外拓改善**：第三方在管面积占比升至 **42.7%**（2025AR p39），独立性缓慢增强。

### 17.1b 不买理由（反方，现价 3.77 HKD）

1. **估值未达龟龟门槛**：剔净现金 FCF **5.2 倍**、股息率 **5.0%**，均未满足熊市 **3 倍 / 6%** 标准。
2. **盈利拐点未确认**：2025 归母净利 **−9.7%**（B02），毛利率降至 **15.0%**（A04）。
3. **关联方依赖仍高**：在管面积 **57.3%** 来自中建/中海系（2025AR p39）。
4. **风险溢价不足**：在 3.77 HKD 起点下，年化期望回报约 **0.1%**，低于国债收益率。

### 17.2 承认的风险

1. **地产下行持续性超出预期**：中海地产销售若无起色，物管增量将持续萎缩
2. **毛利率压缩不可逆**：人工成本刚性 + 物业费限价 + 外拓竞标压价 → 毛利率可能长期低于 15%
3. **关联方依赖风险**：第三方外拓能否补量存在不确定性
4. **资金低效风险**：62.71 亿现金若长期闲置，将拖累 ROE
5. **汇率风险**：港币股息换算为人民币时受汇率波动影响

### 17.3 纪律承诺

- 不因「便宜」而无限制加仓；仓位上限见下
- 不因短期股价波动而止损/止盈；以基本面变化为唯一操作依据
