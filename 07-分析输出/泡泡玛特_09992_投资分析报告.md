```
所有者收益（狭义）= 归母净利 + PPE折旧 + 无形资产摊销 − CAPEX合计（PPE+无形）
                  = 127.76（A06）+ 3.98 + 1.27 − 11.71（A38）
                  = **121.30 亿元**

其中（2025AR p285 附注7，千元÷100,000）：
  PPE 折旧 = 398,202 千元 = 3.98 亿
  无形资产摊销 = 127,050 千元 = 1.27 亿
  （使用权资产折旧 5.93 亿另列，已体现在租赁负债 A29，不重复加计以免与 IFRS16 双计）
```
> 所有者收益 121.30 亿 > FCF 96.94 亿（B03），差额主要来自非现金折旧加回；再投资需求仍以 CAPEX 11.71 亿（A38）为准。
