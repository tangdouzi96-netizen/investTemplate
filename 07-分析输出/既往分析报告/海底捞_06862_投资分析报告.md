```
     = 138.75亿 - 13亿
     = 125.75亿人民币 ≈ 134亿港元

    = 16.50港元 × 55.74亿股
    = 919.71亿港元 ≈ 850亿人民币

剔除现金FCF倍数 = (市值 - 净现金) / FCF
              = (850亿 - 125.75亿) / 54.23亿
              = 724.25亿 / 54.23亿
              ≈ 13.4倍
```
