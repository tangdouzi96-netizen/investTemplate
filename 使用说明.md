# 持仓追踪看板 - 使用说明

本工具自动从互联网获取您持仓股票的最新行情数据，生成一个可实时查看的网页看板。无需手动刷新数据，一键更新即可。

---

## 1. 快速开始

本目录下有 3 个 bat 文件，按顺序使用：

| 步骤 | 文件 | 说明 | 需要联网 |
|------|------|------|----------|
| 第 1 步 | `一键安装.bat` | 安装运行环境（Python/Node.js 依赖） | 是 |
| 第 2 步 | `每日更新.bat` | 从网络获取最新股价数据 | 是 |
| 第 3 步 | `查看看板.bat` | 启动网页看板，浏览器自动打开 | 否 |

**第一次使用**：依次运行 1 -> 2 -> 3。

**日常使用**：先运行「每日更新.bat」，再运行「查看看板.bat」。

> 注意：「查看看板.bat」启动的服务不会自动关闭。看完后在命令行窗口按 `Ctrl+C` 停止即可。

## 2. 自动化（可选）

如果您希望每天定时自动更新数据，可以使用 Windows 任务计划程序。

### 设置步骤

1. 按 `Win+R`，输入 `taskschd.msc`，回车打开"任务计划程序"。
2. 右侧点击"创建基本任务"。
3. 名称填：`持仓追踪-每日更新`。
4. 触发器选"每天"，设置一个合适的时间（建议**收盘后 30 分钟**，如 15:30）。
5. 操作选"启动程序"，浏览选择本目录下的 `每日更新.bat`。
6. "起始于"填本目录的完整路径（例如 `D:\Project\investTemplate`）。
7. 完成。

> 提示：A 股 15:00 收盘，港股 16:00 收盘，16:10 之后运行数据最为完整。

### 注意事项

- 任务执行时电脑需处于**开机且联网**状态。
- 如果电脑设置了休眠/睡眠，任务可能不会触发。可在"条件"选项卡中勾选"唤醒计算机运行此任务"。
- 自动任务只会更新数据，不会自动打开看板。看数据时手动双击「查看看板.bat」即可。

## 3. 修改买入数据

您的持仓数据保存在 Excel 文件中：

```
港股通-泡泡玛特-沪股通-贵州茅台.xlsx
```

### 如何修改

直接用 Excel 打开该文件，修改对应单元格后保存。无需修改脚本或配置文件。

### 各列说明

| 列 | 标题 | 说明 | 示例 |
|----|------|------|------|
| A | 市场 | 填"港股通"或"沪股通" | 港股通 |
| B | 公司名 | 股票名称 | 泡泡玛特 |
| C | 股票代码 | 带后缀的代码 | 09992.HK |
| E | 买入股价 | 您买入时的成交价（每股） | 150.10 |
| F | 买入 PE | 买入时的 PE(TTM) 估值 | 25.3 |
| H | 买入净利润 | 买入时参考的年报净利润（亿元） | 23.5 |
| I | 买入 FCF | 买入时参考的年报自由现金流（亿元） | 18.2 |
| L | 买入预期增速 | 您买入时代入的盈利增长预期（如 15%） | 15 |
| N | 买入预期收益 | 您买入时估算的预期年化回报率 | 12 |
| P | 买入逻辑 | 一句话说明为什么买（看板展示用） | IP 护城河+海外扩张 |
| Q | 护城河 | 竞争优势关键词 | 设计+供应链+渠道 |
| R | 风险因素 | 需要持续关注的潜在风险 | 行业竞争加剧 |

> 提示：
> - D、G、J、K、M、O 列为中间计算列或留白，**不建议手动修改**，脚本会自动重算。
> - 如果您使用 WPS 打开，列号同样是 A、B、C…，对应关系不变。
> - 修改后保存文件，下次运行「每日更新.bat」时新数据即可生效。

### 添加新股

在空白行填入第 3-5 行格式相同的 A~C、E~F、H~I、L、N、P~R 列数据即可。脚本会自动识别所有有代码的行。

### 删除股票

清空该行的股票代码（C 列）即可，或者直接删除整行。

## 4. 数据来源

| 数据项 | 数据源 | 可靠性 |
|--------|--------|--------|
| A 股实时股价 | akshare（东方财富） | 高 - 实时行情 |
| 港股实时股价 | yfinance（雅虎财经） | 高 - 实时行情 |
| PE(TTM) | 同上各自数据源 | 中 - 各平台计算口径略有差异 |
| 净利润 / FCF | 您 Excel 中填写的年报数据 | 每半年更新年报后手动录入 |
| 隐含增速 | 脚本根据 Gordon 模型自动计算 | 仅供参考，含模型假设偏差 |

> 股价和 PE 为实时数据，净利润/FCF 为静态年报数据。两者混合计算的"隐含增速"仅为估值参考，不构成投资建议。

### 数据交叉验证

脚本会自动用两种数据源交叉验证：
- A 股：akshare（东方财富）提供股价和 PE
- 港股：akshare 和 yfinance 双重获取，若数据不一致会在日志中提示

## 5. 常见问题

### Q：双击 bat 文件闪退，看不到错误信息

A：这是 bat 脚本的常见现象。解决办法：

1. 打开资源管理器，进入本目录。
2. 在地址栏输入 `cmd` 回车，打开命令行窗口。
3. 在命令行中直接输入 bat 文件名（如 `每日更新.bat`）回车运行。
4. 这样闪退后也能看到完整的错误提示。

### Q：Python 脚本报错"网络超时"或"连接失败"

A：a股数据（akshare）和港股数据（yfinance）都需要稳定的网络环境。

- 请确认电脑可以正常上网（打开浏览器测试）。
- akshare 的数据接口有时会被限制频率，等待 2-3 分钟后重试。
- yfinance（雅虎财经）在国内访问偶尔不稳定，可以换个时间段再试。
- 如果长期失败，可能需要配置代理。联系技术支持。

### Q：更新失败，提示 akshare 版本过旧

A：A 股数据源 akshare 更新频繁，偶尔接口变动需要升级。

```bash
pip install akshare --upgrade
```

升级后重新运行「每日更新.bat」即可。

### Q：看板页面打开后显示"数据加载失败"

A：通常是因为 `public/data/position_tracker.json` 文件缺失或格式错误。

1. 先确认运行过「每日更新.bat」且提示成功。
2. 如果更新成功但仍加载失败，按下 `F12` 打开浏览器开发者工具 -> Console 标签页，查看红色报错信息，截图发给技术支持。

### Q：港股市值显示为 0 或异常

A：港股实时数据来自 yfinance，此前有部分港股通标的在 yfinance 上的 PE 字段缺失。脚本已经做了容错处理，显示为"--"表示数据不可用，不影响其他数据的正常展示。

### Q：Excel 修改后看板没有变化

A：修改 Excel 后需要**重新运行「每日更新.bat」**，脚本才会把 Excel 数据与新行情数据合并输出。看板显示的是脚本生成的最新 JSON 文件，不会自动读取 Excel。

### Q：能用手机看吗

A：VitePress 开发服务器默认只监听本地。在同一局域网内的手机/平板可以访问：

1. 先运行「查看看板.bat」。
2. 在命令行输出中找到 `Network` 那一行的地址（类似 `http://192.168.x.x:5173/`）。
3. 在手机浏览器中输入该地址即可查看。

> 注意：手机和电脑必须在**同一 Wi-Fi 网络**下。手机浏览器地址栏输入完整链接，包括冒号和端口号。
