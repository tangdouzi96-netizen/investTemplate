- **类型**: 港股/A股个股投资分析模板系统

### 快速筛选框架（分析前强制执行）⭐

> **核心原则**：已分析过的直接更新，没分析过的全部先筛再问。

#### 分析入口规则

| 场景 | 处理 |
|------|------|
| 标的在"已分析标的"列表中 | ⬇ 跳过筛选，直接更新报告 |
| 标的**不在**列表中（新标的） | ⬇ 先执行 6 关筛选，输出结果，**询问是否继续**，等用户确认 |

#### 6 关定量筛选

| # | 维度 | 通过标准 | 不通过处理 |
|---|------|---------|-----------|
| 1 | **护城河代理**（三选二） | ① 毛利率 5 年稳定或上升 + 行业前五 ② ROIC 5 年均值 > 12% + 行业前三 ③ 市占率行业前三 | ❌ 三项全不满足 → 筛掉 |
| 2 | **盈利质量** | FCF/净利润 5 年均值 > 70% | ❌ < 70% → 筛掉 |
| 3 | **管理层红线**（必须全过） | ① ROE 5 年均值 > 12%（非金融股）② 近 3 年无财务造假/违规处罚 ③ 大股东近 3 年未大规模减持（>5%） | ❌ 任一项不通过 → 筛掉 |
| 4 | **财务安全** | 净现金 - 有息负债 > 0 | ❌ 不通过 → 筛掉（REITs/金融/基建行业豁免此条） |
| 5 | **股息可持续** | ① 连续 5 年分红 ② 分红率 ≤ 80%（不分过头）③ 股息率 > 4%（A 股）/ > 6%（港股） | ❌ 任一项不通过 → 筛掉 |
| 6 | **估值闸门** | 剔除净现金 FCF 倍数 < 3 倍（熊市）/ 或 PE < 15 倍（A 股） | ❌ 不通过 → 筛掉 |

#### CAPEX 预警（不直接筛掉，配合 ROIC 分档判断）

```
CAPEX/经营现金流（5 年平均）
├── < 30%   → ✅ 绿灯
├── 30-50%  → 🟢 正常维持型
├── 50-70%  → 🟡 预警：必须配合 ROIC 判断
│              ROIC > 15% → 扩张期投入，可接受
│              ROIC < 10% → 低效投资，深度质疑
├── > 70%    → 🔴 严重预警：回溯 5 年 CAPEX 趋势
│              ROIC 持续走低 → 价值陷阱信号
│              ROIC 稳定高企 → 行业特性使然，接受
```

```
## 快速筛选结果：{公司名称}（{代码}）

| # | 维度 | 结果 | 详情 |
|---|------|------|------|
| 1 | 护城河代理 | ✅/❌ | 通过项：xxx |
| 2 | 盈利质量 | ✅/❌ | FCF/NP 5年均值：xx% |
| 3 | 管理层红线 | ✅/❌ | ROE：xx%，违规：无，减持：无 |
| 4 | 财务安全 | ✅/❌ | 净现金：xx亿 |
| 5 | 股息可持续 | ✅/❌ | 分红年数：x年，分红率：xx%，股息率：xx% |
| 6 | 估值闸门 | ✅/❌ | 剔除净现金FCF倍数：x.x倍 |
| ⚠️ | CAPEX预警 | 正常/🟡预警/🔴严重 | CAPEX/OCF：xx%，ROIC：xx% |

**结论**：✅ 通过筛选 / ❌ 未通过（卡在第 X 关：{具体原因}）
```

### 生成投资分析报告流程

```
0. 快速筛选（6关定量检查 + CAPEX预警）← 新标的强制执行，已分析/豁免标的跳过
   → 未通过 → 告知用户卡在哪一关及具体原因，停止分析
   → 通过   → 继续
1. 读取个股分析标准模版.md（获取最新版本和结构）
2. 搜索标的最新财务数据（年报、股价、行业数据）
3. 按改进框架（V5.5.14 + 7大缺口）生成报告
4. 报告输出 → 07-分析输出/{公司}_{代码}_投资分析报告.md
5. 更新 07-分析输出/index.md 目录索引
6. 告知用户完成，由用户决定是否 git commit + push
```
